TRANSCRIPCIONES
===============

Estas transcripciones potencian el AI SEO (ChatGPT, Perplexity y Claude citan
mejor las páginas con texto completo). Cada archivo {videoId}.txt corresponde
a un episodio y se muestra solo en su página.

MÉTODO AUTOMÁTICO (recomendado)
-------------------------------
Un proceso de GitHub Actions baja las transcripciones solo, usando Supadata.
No hay que copiar y pegar nada. Ver la guía de configuración en el README
principal del proyecto (sección "Transcripciones automáticas"). Una vez
configurado, corre solo cada semana y agrega las de los episodios nuevos.

MÉTODO MANUAL (opcional / respaldo)
-----------------------------------
Si alguna transcripción no se pudo bajar sola, o querés editarla:
1. ID del video = lo que va al final de mateyeventos.com/episodios/XXXXXXXXXXX
2. Copiá el texto desde YouTube Studio o desde "Mostrar transcripción" en YouTube.
3. Creá/editá el archivo content/transcripts/XXXXXXXXXXX.txt y pegá el texto.
